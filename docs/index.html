Overview

HYDRACTRL is a specialized tool for live visual performances built on top of Hydra-Synth, a JavaScript library created by Olivia Jack. HYDRACTRL enhances the Hydra experience with a performance-oriented interface, scene management, MIDI integration, and more.

Hydra allows you to create stunning visual patterns and animations through code. By combining oscillators, geometry transformations, and color manipulations, you can create complex, hypnotic visuals in real-time that react to various inputs including audio.

Scene Management

Store and recall up to 64 scenes (4 banks of 16 slots) for quick access during performances.

MIDI Integration

Control your visuals using Korg nanoPad MIDI controller.

Code Editor

Syntax highlighting and customizable themes for easier live coding.

Multiple Themes

5 built-in themes including the default dark theme, light theme, high contrast, neon 80s, and 90s pop.

Breakout View

Send your visuals to a second screen or window with exact size control for recordings or projections.

Import/Export

Save and share your scene banks with thumbnail previews for easy organization.

Key Features

Scene Storage

HYDRACTRL provides a comprehensive scene management system with 4 banks of 16 slots each, giving you quick access to 64 different visual presets during your performance. Each scene stores:

  • The complete Hydra code that generates your visuals (Base64 encoded)
  • A visual thumbnail for easy recognition
  • Automatic persistence between sessions

Advanced Editor

The integrated code editor is optimized for live coding scenarios:

  • JavaScript syntax highlighting with Hydra function recognition
  • Error reporting with detailed messages
  • Adjustable window size and position
  • Keyboard shortcuts for faster workflow

MIDI Controller Support

Connect a MIDI controller to trigger scenes during performance:

  • Optimized for Korg nanoPAD and similar pad controllers
  • Scene bank synchronization with nanoPAD scenes
  • Trigger any of the 16 slots in the current bank with MIDI pads

Breakout Window

Send your visuals to a secondary window with precise dimension control:

  • Predefined dimensions for common resolutions (720p, 1080p, etc.)
  • Perfect for recording, streaming, or projecting your work
  • Synchronized content between main and breakout views

Keyboard Shortcuts

HYDRACTRL provides numerous keyboard shortcuts to streamline your workflow during performances:

Shortcut Function
Ctrl + Enter Run the current code
Ctrl + S Save code to the active slot
Ctrl + X Export scene bank
Ctrl + I Import scene bank
Escape Toggle editor visibility
Ctrl + 1-9 Select slots 1-9 in the current bank
Ctrl + ← / Ctrl + → Cycle between banks

Built With

HYDRACTRL combines modern web technologies to create a powerful live coding environment:

Hydra-Synth

The foundation of HYDRACTRL is Hydra-Synth, a powerful JavaScript library for live coding visuals created by Olivia Jack. Hydra uses WebGL to create stunning visual patterns and is inspired by analog modular synthesizers.

JavaScript & Bun

Built with modern JavaScript and leveraging Bun for improved performance, HYDRACTRL offers a responsive experience even with complex visuals.

CodeMirror 6

The code editor is powered by CodeMirror 6, providing syntax highlighting, error reporting, and an optimized editing experience for live performances.

Web MIDI API

MIDI integration is implemented using the Web MIDI API, allowing connectivity with hardware controllers for tactile control during performances.

Getting Started

Launch the App

To start using HYDRACTRL right away, simply click the "Visit HYDRACTRL" button at the top of this page.

Basic Workflow

  1. Write your Hydra code in the editor (or start with one of the examples)
  2. Press Ctrl+Enter to run the code and see the visuals
  3. Use Ctrl+S to save your creation to the currently selected slot
  4. Build a library of scenes by saving to different slots
  5. Use keyboard shortcuts or MIDI controller to switch between scenes during performance

Local Development

To run HYDRACTRL locally or contribute to its development:

  1. Clone the repository: git clone https://github.com/dxviie/HYDRACTRL.git
  2. Install dependencies: bun install
  3. Start development server: bun dev
  4. Build for production: bun run build
